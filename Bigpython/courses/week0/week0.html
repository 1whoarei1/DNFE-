Step 1: Download the Full Installer Follow these steps to download the full installer:

1. Open a browser window and navigate to the Python.org Downloads page for Windows.

2.Under the “Python Releases for Windows” heading, click the link for the Latest Python 3 Release - Python 3.x.x. As of this writing, the latest version was Python 3.8.4.

3 to the bottom and select either Windows x86-64 executable installer for 64-bit or Windows x86 executable installer for 32-bit. If you aren’t sure whether to select the 32-bit or the 64-bit installer, then you can expand the box below to help you decide.
